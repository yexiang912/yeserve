## 🎯 项目简介

**YeServe** 是由 **yx** 原创开发的一套专业级Ubuntu服务器自动化部署工具集。项目包含三个不同定位的脚本，采用 `dialog` 工具创建美观的终端图形界面，让服务器部署变得简单直观。

**项目特点**：
- 🎨 **精美终端GUI** - 基于dialog的原创界面设计
- ⚡ **一键部署** - 自动化安装和配置
- 🔧 **模块化设计** - 按需选择功能模块
- 📊 **多版本可选** - 从新手到专业用户全覆盖
- 🚀 **便捷启动器** - v9.5新增启动器，自动获取最新版本

## 🚀 启动器使用说明

### 启动器介绍
v9.5版本引入了启动器，自动从GitHub获取最新版本的脚本：

**启动器特点：**
- 🔄 **自动获取最新版** - 每次运行都获取最新脚本
- 📋 **脚本选择界面** - 图形化选择三个核心脚本
- ⚡ **简单易用** - 一行命令即可使用

### 启动器使用方法
```bash
# 使用启动器（推荐）
sudo wget -O luncher.sh https://raw.githubusercontent.com/yexiang912/yeserve/main/luncher.sh
sudo chmod +x luncher.sh
sudo ./luncher.sh

```

### 启动器工作原理
1. **下载启动器**：从GitHub下载最新启动器脚本
2. **获取脚本**：启动器自动下载最新版本的核心脚本
3. **显示菜单**：提供三个脚本的选择界面
4. **执行脚本**：运行用户选择的脚本

## 📦 三个核心脚本

### 1. **基础版 - `yeserve.sh`**
**定位：快速入门，基础功能 - 适合新手**

- 快速测试环境搭建
- 基础服务器配置

---

### 2. **GUI增强版 - `serveui.sh`**

- 🔧 **基础运维** - 常用工具安装
- ⚙️ **系统优化** - 基础性能优化

**适用场景：**
- 有一定Linux经验的用户
- 个人项目和小型业务
- 需要完整的Docker和面板环境

### 3. **专业版 - `servepro.sh`**

- 🛠️ **高级工具** - 扩展运维工具集合
- 💻 **开发环境** - Node.js/Python/Java/PHP
- 🗄️ **数据库** - MySQL/PostgreSQL/Redis/MongoDB
- 🌍 **Web服务器** - Nginx/Apache
- 📊 **监控系统** - 服务状态监控

**适用场景：**
- 专业运维人员或开发者
- 企业测试和开发环境
- 需要完整开发和生产环境

## 🛡️ 安全配置说明

### 系统要求
- ✅ **操作系统**：Ubuntu 18.04/20.04/22.04/24.04
- ✅ **内存**：至少 1GB RAM（基础版），2GB+（专业版推荐）
- ✅ **磁盘**：至少 10GB 可用空间
- ✅ **网络**：稳定的互联网连接
- ✅ **权限**：root 或 sudo 权限

### 安全建议
1. **系统更新**：定期更新操作系统和安全补丁
2. **数据备份**：重要数据定期备份到外部存储
3. **权限管理**：合理分配用户权限，避免使用root账户
4. **网络防护**：配置防火墙，仅开放必要端口
